# Design System

## Overview
This document defines the visual language of the project.

## Tokens
- **Colors**:
  - `primary`: `#136dec` (Vibrant Blue)
  - `primary-soft`: `rgba(19, 109, 236, 0.1)`
  - `text-heading`: `#0d131b`
  - `text-subtle`: `#4c6c9a`
  - `background-light`: `#f6f7f8`
  - `background-dark`: `#101822`

- **Typography**:
  - `display`: "Space Grotesk", sans-serif (Precision weighting)
  - `sans`: "Space Grotesk", sans-serif

- **Radii**:
  - `default`: `0.25rem`
  - `lg`: `0.5rem`
  - `xl`: `0.75rem`
  - `2xl`: `1rem`
  - `3xl`: `1.5rem`
  - `full`: `9999px`

- **Spacing**: 
  - Standardized on 4px/8px grid system.
